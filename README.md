# aprendiendo-python
# 🎮 Análisis de Videojuegos más Vendidos

## ¿Qué hace este proyecto?
Análisis de los 50 videojuegos más vendidos de la historia usando Python y pandas.

## ¿Qué aprendí?
- Cargar y explorar datos con pandas
- Funciones con parámetros y return
- groupby, sort_values, idxmax, value_counts
- Gráfica de barras con matplotlib
- Recorrer un dataframe con for e iterrows()
- Clasificar datos con if/elif/else
- Recibir entrada del usuario con input()

## Tecnologías usadas
- Python
- pandas
- matplotlib
- Google Colab

## Funciones implementadas
1. `juegos_x_genero()` — cuántos juegos hay por género
2. `juego_mas_vendido()` — el juego con más ventas globales
3. `genero_mas_ventas()` — el género con más ventas en total
4. `juegos5masvendidos()` — top 5 juegos más vendidos
